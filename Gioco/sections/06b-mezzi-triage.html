Il Triage: Decidere in Pochi Secondi

Il metodo START classifica rapidamente le vittime. Clicca su ogni passo per seguire l'algoritmo. N.B.: per i soccorritori MSB il codice NERO è sostituito dal ROSSO.

Passi del Triage S.T.A.R.T. (Semplificato)

Clicca sul passo evidenziato per rivelare il successivo.

1. Cammina? → Se SÌ: VERDE. Se NO, vai al passo 2.
2. Respira? (dopo apertura vie aeree) → Se NO: ROSSO. Se SÌ, vai al passo 3.
3. Freq. Respiratoria > 30/min? → Se SÌ: ROSSO. Se NO, vai al passo 4.
4. Perfusione? polso radiale assente → Se SÌ: ROSSO. Se NO, vai al passo 5.
5. Esegue ordini semplici? → Se NO: ROSSO. Se SÌ: GIALLO.

Distribuzione dei codici
